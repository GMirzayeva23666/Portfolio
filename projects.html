Logic Gates Simulator

Frontend • Interactive • Education

An interactive web-based simulator for digital logic gates. This project allows users to build and test digital circuits using basic logic gates like AND, OR, NOT, XOR, and more. Built with HTML5 Canvas and JavaScript, it provides real-time visualization of circuit behavior.

Features include drag-and-drop gate placement, wire connections, truth table generation, and step-by-step simulation. Perfect for students learning digital electronics and computer science fundamentals.

Technology Stack

HTML5, CSS3, JavaScript, Canvas API

Features

Interactive Simulation • Real-time Feedback • Educational

Impact

Used by 500+ students • Open-source educational tool
